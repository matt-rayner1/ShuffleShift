# ABBREVIATIONS 

## Complexity theory 
P = Polynomial time 
NP = Nondeterministic Polynomial time 

## Language/Grammar
CFG = Context-Free Grammar 
    Rules describing sentence structure 
    e.g: Det Adj N V Prep Det N 
         The big dog pees on the tree

POS = Part Of Speech 
    Noun, verb, adjective etc 

NP = Noun Phrase 
VP = Verb Phrase 
Det = Determiner 
Adj = Adjective 
Prep = Preposition 

## Algorithms 
DFS = Depth-First Search 
CSP = Constraint Satisfaction Problem 
Memoisation = Caching failed or solved subproblems 
Heuristic = Rule of thumb to guide search
